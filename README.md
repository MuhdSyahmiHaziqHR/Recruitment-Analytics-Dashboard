# Recruitment Analytics Dashboard

A recruitment analytics project developed using Microsoft Excel and Power BI to automate candidate screening, ranking, and recruitment insights.

---

## Project Overview

The Recruitment Analytics Dashboard is designed to support data-driven recruitment decisions by organizing candidate information, calculating candidate scores, ranking applicants, and providing interactive dashboards for HR professionals.

This project simulates a real-world recruitment process and demonstrates how HR analytics can improve hiring efficiency and decision-making.

---

## Objectives

- Automate candidate screening and ranking.
- Analyze candidate information efficiently.
- Generate recruitment insights and reports.
- Visualize hiring trends using dashboards.
- Support HR decision-making through data analytics.

---

## Features

### Candidate Database
Stores candidate information including:

- Candidate ID
- Name
- Degree
- CGPA
- Experience (Years)
- Excel Skill Level
- Interview Score

### Candidate Scoring System
Candidates are evaluated using weighted criteria:

| Criteria | Weight |
|-----------|---------|
| CGPA | 25% |
| Experience | 25% |
| Excel Skill | 20% |
| Interview Score | 30% |

### Candidate Ranking System
- Automatic ranking of candidates.
- Identification of top-performing applicants.
- Shortlist, Review, and Reject recommendations.

### Recruitment Dashboard
Provides key recruitment metrics:

- Total Applicants
- Average Candidate Score
- Top Candidate Score
- Number of Shortlisted Candidates
- Number of Rejected Candidates
- Number of Review Candidates

### Candidate Profile Analysis
Provides:

- Recommendation distribution
- Candidate score comparison
- Top candidate ranking
- Interactive filtering and analysis
